# bode

Scape venues that I frequent for concerts and find the musicians and bands that I listen to.

## Requirements

##### Ruby 2.3

Binaries:

    chromium
    chromium-driver
    
Ruby gems:

    nokogiri
    capybara
    selenium-webdriver
    rainbow
